# Community Projects

Here is a list of PicoLimbo related community projects:

## MinecraftDataDump
A Minecraft mod for dumping various data from the game during run-time.
Used to generate reports, allowing players to join PicoLimbo using modded clients.

See [MinecraftDataDump on GitHub](<https://github.com/tyzeron/MinecraftDataDump>).

> [!NOTE]
> Mod support for PicoLimbo is still work in progress and not available yet.
> For now, this mod is used for development purposes only.

## ThirdPlaceMcPicoLimboWrapper
A Java wrapper application that automatically downloads and launches the latest PicoLimbo Minecraft limbo server binary with support for running multiple server instances.
Allows you to start PicoLimbo on hosts that don't allow anything other than Java applications.

See [ThirdPlaceMcPicoLimboWrapper on GitHub](<https://github.com/lerndmina/ThirdPlaceMcPicoLimboWrapper>).

> [!NOTE]
> PicoLimbo also has an official Java Wrapper available on [Modrinth](https://modrinth.com/plugin/picolimbo-java-wrapper).
